Product

PhishGuard AI is a real-time phishing detection platform that analyzes website URLs using machine learning to identify malicious and zero-day phishing threats before users interact with them.

Features

  • ⚡ Real-time phishing detection
  • 🧠 AI-based URL & domain analysis
  • 📊 Risk scoring instead of simple blocking
  • 🛡️ Zero-day phishing protection
  • 🌐 Web app & browser-extension ready

Documentation

The system exposes a FastAPI-based REST API that accepts URLs and returns phishing predictions with a confidence score.
